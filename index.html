Supabase Database

Connect to Supabase for persistent cloud storage. Without it, all data resets on server restart.
Find in: Supabase Dashboard → Project Settings → API → Project URL
Find in: Project Settings → API → anon / public key
📋 Setup steps:
1. Create free project at supabase.com
2. Go to SQL Editor → paste the schema from http://localhost:3002/api/schema
3. Paste your Project URL and anon key above → click Save & Connect

⚡ Or use env vars (production):
SUPABASE_URL=https://xxxx.supabase.co   SUPABASE_KEY=eyJhbGci...
